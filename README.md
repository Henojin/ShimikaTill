# .NET Framework 4が必要です
.NET Freamwork 4はWindows XP等の古いOSには標準搭載されていないため、<br>
https://www.microsoft.com/ja-jp/download/details.aspx?id=17718<br>
こちらからインストールして利用してください。

開発する際、Visual Studio 2022だと標準で対応していないため、<br>
